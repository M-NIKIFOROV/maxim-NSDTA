# Metadata

This directory contains metadata files describing the datasets and variables.

## Expected Files
- `data_dictionary.csv`: Variable names, descriptions, and types
- `sector_metadata.csv`: Information about each sector
- `source_documentation.md`: Detailed information about data sources

## Purpose
- Document variable definitions and coding schemes
- Track data provenance and collection methods
- Maintain data quality documentation

## Notes
- Keep metadata updated as data structure changes
- Reference this directory in analysis scripts for clarity
