## Project: NSDTA Data Pipeline

This repository contains the NSDTA (National Sector Data and Trend Analysis) data pipeline project.

### Overview
The NSDTA project implements a comprehensive data analysis pipeline for sector data (education, health, and law & justice) with three distinct phases:

1. **Data Loading & Cleaning (R)**: Load raw sector data, clean and standardize it, export to CSV
2. **Schema Organization (SQL)**: Import data into SQLite, construct variables, generate descriptive statistics
3. **Statistical Inference (R)**: Connect to database, perform t-tests and regressions, create visualizations
